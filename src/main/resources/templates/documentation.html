добавление записи

https://lists-regions.herokuapp.com/rest-api/insertValue?regionName=Республика%20Башкортостан®ionShortName=BA&flag=https://ruxpert.ru/Файл:Flag_of_Bashkortostan.png

Примечание
В случае добавления записи через строку браузера, в качестве пробела требуется использовать символ: %20

пример ответа

the Республика Башкортостан was added

 удаление записи

https://lists-regions.herokuapp.com/rest-api/delete?shortName=test

Примечание
Удаляется элемент по короткому имени региона, пример "IN"

пример ответа

the test was deleted
